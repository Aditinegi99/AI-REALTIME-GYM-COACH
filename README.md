# 🏋️ AI Real-Time Gym Coach
An AI-powered virtual fitness coach that provides real-time exercise tracking, posture correction, voice guidance, workout history, and performance analytics using Computer Vision and Machine Learning. The system helps users exercise with proper form, track their progress, and stay motivated through interactive fitness experiences.

# Why This Project?
Most fitness applications either track workouts or provide static guidance. AI Real-Time Gym Coach combines real-time pose estimation, voice coaching, posture correction, workout analytics, and an immersive warm-up experience into a single intelligent fitness platform, creating a more engaging and personalized workout experience.


# 🚀 Features
- 🎯 Real-time pose detection and body tracking
- 🔢 Automatic repetition counting
- ✅ AI-powered posture and form correction
- 🎙️ Voice-assisted workout guidance
- 🎮 Interactive immersive warm-up experience
- 📊 Live workout analytics and performance tracking
- 📈 Workout history and progress monitoring
- 💪 Support for multiple exercises
- ⚡ Instant feedback on exercise technique
- 🧠 Exercise-specific movement analysis
- 🖥️ Interactive and user-friendly interface


# 🎮 Interactive Warm-Up Experience
Transforms traditional warm-ups into an engaging game-like activity that prepares users physically and mentally before beginning their workout session.

# 🎙️ Voice Coaching
The AI coach provides voice-based instructions and feedback during workouts, helping users maintain proper form and stay focused without constantly looking at the screen.

# 📊 Workout Analytics & History
- Track completed workout sessions
- Monitor repetition counts and performance
- View historical workout data
- Analyze progress over time
- Maintain consistency through performance insights

# ⚙️ How It Works
1. Launch the application.
2. Complete the immersive warm-up session.
3. Select an exercise.
4. The camera captures body movements in real time.
5. AI detects body landmarks and analyzes posture.
6. Repetitions are counted automatically.
7. Voice feedback guides the user throughout the workout.
8. Session data is saved for future analysis and progress tracking.
